# 🚀 Next.js Projects Collection

A single repository showcasing two modern **Next.js** projects, deployed on **Vercel**, demonstrating frontend UI development and API integration.

## 📌 Live Projects

| Project | Description | Live URL |
|-------|------------|----------|
| **Next.js Books API** | A Next.js application demonstrating API routes, server-side data fetching, and dynamic rendering of book data. | https://nextjs-books-api.vercel.app/ |

## 📦 Project 1: Next.js Books API

### Overview

This project demonstrates how to build and consume backend APIs using **Next.js API routes** with optimized server-side rendering.

### Key Features

- API routes inside Next.js  
- Server-side data fetching  
- Clean JSON responses  
- Optimized rendering  

---

## 👨‍💻 Project 2: Next.js Assignments / Portfolio

### Overview

A frontend-focused Next.js project inspired by a Figma design. It presents structured sections such as profile information, projects, skills, and assignments.

### Key Features

- Responsive UI layout  
- Component-based structure  
- Modern and clean design  
- Vercel deployment

```bash
git clone <repository-url>

npm install
npm run dev
```

Open in browser:

```
http://localhost:3000
```
